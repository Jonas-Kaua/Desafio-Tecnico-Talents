## DESCRICAO DO PROJETO ##

Projeto realizado para o teste técnico da Consultoria Talents, nele foi desenvolvido
um CRUD de uma livraria virtual contendo livros, dentro suas operações desenvolvidas temos

* Listar Livros, Cadastrar Livro, Excluir Livro, Editar Livro, Buscar Livro por filtro
de título, código ou código do autor, e uma paginação mockada. 



## TECNOLOGIAS UTILIZADAS ##

FrontEnd : React, Bootstrap, Reactstrap, NodeJs

BackEnd : Nodejs, Express, Sequelize e Cors

Banco de Dados : MySql



## INSTRUÇÕES PARA INSTALAÇÃO ##

- FRONTEND

abra a pasta do frontend no vs code ou em outra IDE de execução de projetos
rode os comandos no terminal :

"yarn install"(recomendado) ou "npm install" para instalação de dependencias

após isso para rodar o front rode o comando :

"yarn dev"(recomendado) ou "npm start"

- BACKEND

abra a pasta do backend no vs code ou em outra IDE de execução de projetos
rode os comandos no terminal :

"yarn install"(recomendado) ou "npm install" para instalação de dependencias

após isso para rodar o back rode o comando :

"yarn dev"(recomendado) ou "npm start"


- BANCO DE DADOS MYSQL

rode o arquivo 'livraria.sql' presente na pasta mãe do projeto em alguma IDE 
do sql de sua preferência para criar o Esquema com a tabela utilizada no projeto 
no seu MySql, a IDE que eu utilizei foi o MySql WorckBench 8.0CE
após isso suba o mysql utilizando um programa que suporte lançar o serviço no ar,
no meu caso eu utilizei o XAMPP Control Pane v3.3.0
